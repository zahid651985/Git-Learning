Introduction of CSS

In this section, we will look at how to make your web pages moreattractive,controlling the design of them using CSS.

CSS Associates Style rules with HTML element

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. A CSS rule contains two parts: a selector and a declaration

CSS Properties Affect How Elements Are Displayed

CSS declarations sit inside curly brackets and each is made up of two parts:a propertyand a value, separated by a colon. You can specify several properties in one declaration, each separated by a semi-colon.
